Creating custom Pignus Images

To create Pignus distribution images we use appliance-creator, a tool uses kickstart files to describe the installation package set and configuration. We keep our kickstarts in pignus-kickstarts repository.

Running appliance-creator from your existing Pignus installation is as simple as this: 

$ su
$ git clone https://github.com/pignus-project/pignus-kickstarts 
$ cd pignus-kickstarts
$ su
# dnf -y install appliance-creator
# appliance-creator -n pignus-zero -c pignus-zero.ks --no-compress
#

Tip: you can also run appliance-creator from a Fedora/arm installation on a different board than Raspberry Pi. Building on a non-ARM machine (such as Intel-based PC) won't work.
