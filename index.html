Pignus. Raspberry Pi Linux the way it should be

We love Raspberry Pi. It revolutionized the way we think about computing. A board like no other. A universally available Linux computer, loved by the hackers and makers worldwide.

We also love Linux. We believe there's tremendous value in its development model: Sharing as much as possible. Sticking close to the mainline development. Committing to industry standard tools. This provides good insurance your work stays relevant and secure. With Raspberry Pi's default software stack diverging too much from the world we love and trust, we though there's gap to fill. We're doing that now. Enter Pignus:

  • Optimized for ARMv6, BCM2835 architecture
  • Uses mainline Linux kernel
  • Boots with U-Boot
  • RPM-based

Pignus runs on BCM2835-based boards: Raspberry Pi Model A, Raspberry Pi Model A+, Raspberry Pi Model B and Raspberry Pi Zero boards. Raspberry Pi 2 Model B, is not supported by the mainline kernel yet. We recommend you check out Fedora once it is.

Pignus Zero

Plug in one USB cable and you're done.

Pignus zero provides a virtual serial port and a network device via USB. An ideal way to get your RPi Zero running in no time!

You can download an Alpha release for testing now. Please note it's a pre-release quality software and shouldn't be used in production.
